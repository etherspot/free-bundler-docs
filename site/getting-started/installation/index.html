    Installation Guide¶

    This guide will help you install and set up the necessary tools and dependencies for working with EntryPoint v0.8 and UserOperations.

    Prerequisites¶

    Before installing, ensure you have:

    • Node.js: Version 16 or higher
    • npm or yarn: Package manager
    • Git: Version control
    • Ethereum RPC Access: Infura, Alchemy, or local node

    Package Installation¶

    Core Dependencies¶

    Install the essential packages for EntryPoint v0.8 development:

    # Using npm
npm install @account-abstraction/sdk ethers

# Using yarn
yarn add @account-abstraction/sdk ethers

# Using pnpm
pnpm add @account-abstraction/sdk ethers

    Additional Utilities¶

    For enhanced functionality, consider installing:

    # TypeScript support
npm install -D typescript @types/node

# Testing framework
npm install -D jest @types/jest

# Development tools
npm install -D hardhat @nomicfoundation/hardhat-toolbox

    Environment Setup¶

    1. Create Project Structure¶

    mkdir entrypoint-v8-project
cd entrypoint-v8-project
npm init -y

    2. Install Dependencies¶

    npm install @account-abstraction/sdk ethers

    3. Environment Configuration¶

    Create a .env file:

    # .env
RPC_URL=https://sepolia.infura.io/v3/YOUR_PROJECT_ID
PRIVATE_KEY=your_private_key_here
ENTRYPOINT_ADDRESS=0x5FF137D4b0FDCD49DcA30c7CF57E578a026d2789
BUNDLER_URL=https://api.stackup.sh/v1/bundler

    4. TypeScript Configuration (Optional)¶

    Create tsconfig.json:

    {
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "lib": ["ES2020"],
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}

    SDK-Specific Installation¶

    @account-abstraction/sdk¶

    The official Account Abstraction SDK:

    npm install @account-abstraction/sdk

    Features: - UserOp construction utilities - EntryPoint integration - Bundler communication - TypeScript support

    Alternative SDKs¶

    userop.js¶

    npm install userop

    ethers.js with custom integration¶

    npm install ethers

    Development Environment¶

    Local Development¶

    For local development and testing:

    # Install Hardhat for local blockchain
npm install -D hardhat @nomicfoundation/hardhat-toolbox

# Initialize Hardhat
npx hardhat init

    Testnet Configuration¶

    Configure for testnet development:

    // hardhat.config.ts
import { HardhatUserConfig } from "hardhat/config";
import "@nomicfoundation/hardhat-toolbox";

const config: HardhatUserConfig = {
  solidity: "0.8.19",
  networks: {
    sepolia: {
      url: process.env.RPC_URL,
      accounts: [process.env.PRIVATE_KEY!]
    }
  }
};

export default config;

    Bundler Setup¶

    Using Public Bundlers¶

    No additional setup required for public bundlers:

    • Stackup: https://api.stackup.sh/v1/bundler
    • Pimlico: https://api.pimlico.io/v2/sepolia/rpc
    • Etherspot: https://api.etherspot.io/v1/bundler

    Self-Hosted Bundler¶

    For production or custom requirements:

    # Clone bundler repository
git clone https://github.com/stackup-wallet/stackup-bundler.git
cd stackup-bundler

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your configuration

# Start bundler
npm start

    Verification¶

    Test Installation¶

    Create a simple test file to verify your setup:

    // test-setup.ts
import { ethers } from 'ethers';

async function testSetup() {
  try {
    // Test provider connection
    const provider = new ethers.JsonRpcProvider(process.env.RPC_URL);
    const network = await provider.getNetwork();
    console.log('Connected to network:', network.name);

    // Test EntryPoint contract
    const entryPointAddress = process.env.ENTRYPOINT_ADDRESS;
    const entryPoint = new ethers.Contract(
      entryPointAddress!,
      ['function getUserOpHash(tuple) view returns (bytes32)'],
      provider
    );

    console.log('EntryPoint contract accessible');
    console.log('Setup complete!');

  } catch (error) {
    console.error('Setup failed:', error);
  }
}

testSetup();

    Run the test:

    npx ts-node test-setup.ts

    Troubleshooting¶

    Common Issues¶

    1. Network Connection Errors¶

    # Check RPC URL
curl -X POST -H "Content-Type: application/json" \
  --data '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}' \
  YOUR_RPC_URL

    2. Package Installation Issues¶

    # Clear cache and reinstall
npm cache clean --force
rm -rf node_modules package-lock.json
npm install

    3. TypeScript Errors¶

    # Install type definitions
npm install -D @types/node @types/jest
